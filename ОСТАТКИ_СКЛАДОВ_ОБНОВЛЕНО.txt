Как использовать:
=================

1. Перейти в раздел "📦 Остатки складов"
2. Выбрать фильтры:
   - Склад (или оставить "Все склады")
   - Товар (или оставить "Все товары")
   - Дата остатков (по умолчанию - сегодня)
3. Установить/снять галочку "Показать нулевые остатки"
4. Нажать "📊 Сформировать отчет"
5. Использовать кнопки:
   - 🖨️ Печать - для печати отчета
   - 📊 Excel - для экспорта в Excel

Особенности:
============

- Отчет группируется по группам складов (warehouse_group)
- Внутри каждой группы склады сортируются по алфавиту
- Для каждого склада показываются товары с остатками
- В конце отчета выводится общий итог по всем товарам
- Показывается информация о расходах за выбранную дату
- Формат полностью соответствует старому проекту
