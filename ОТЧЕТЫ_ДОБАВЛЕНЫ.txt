✅ ОТЧЕТЫ ДОБАВЛЕНЫ

Добавлены 4 новых раздела отчетов:

1. 🚂 СВОД ВАГОНОВ
   - Показывает все вагоны прихода за период
   - Фильтры: период с/по
   - Колонки: Вагон, Дата, Фирма, Склад, Товар, Кол-во по док, Кол-во факт, Разница, Вес (тонн)

2. 📦 СВОД ОСТАТКОВ
   - Показывает остатки по складам и товарам
   - Фильтры: период с/по
   - Расчет: Приход - Расход = Остаток
   - Колонки: Склад, Товар, Приход (тонн), Расход (тонн), Остаток (тонн)

3. 💳 ОТЧЁТ ПО ДОЛГАМ
   - Показывает долги клиентов
   - Фильтры: период с/по
   - Расчет: Расход - Погашено = Долг
   - Колонки: Клиент, Расход ($), Погашено ($), Долг ($)
   - Сортировка: по долгу (больший долг сверху)

4. 📊 ИТОГИ ВАГОНОВ
   - Показывает количество вагонов по складам и товарам
   - Фильтры: период с/по
   - Колонки: Склад, Товар, Количество вагонов, Общий вес (тонн)

ФАЙЛЫ:
- postgres-version/frontend/index.html - HTML разметка разделов
- postgres-version/frontend/js/reports.js - JavaScript логика
- postgres-version/frontend/js/main.js - навигация

ФУНКЦИИ:
- updateWagonSummary() - обновление свода вагонов
- updateBalanceSummary() - обновление свода остатков
- updateDebtReport() - обновление отчёта по долгам
- updateWagonTotals() - обновление итогов вагонов

Все отчеты работают с данными из window.appData и обновляются при переходе в раздел.

Перезапустите сервер и проверьте работу отчетов!
