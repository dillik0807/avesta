# ✅ Управление ценами товаров - ГОТОВО

## Задача:
Добавить возможность указывать цену товара в управлении, и автоматически подставлять её при выборе товара в форме "Расход". Пользователь может изменить цену вручную.

## Что сделано:

### 1. База данных:
- ✅ Добавлено поле `price` в таблицу `products`
- ✅ Создан скрипт миграции `add-product-price.sql`
- ✅ Обновлена схема `schema.sql`

### 2. Backend (`routes/management.js`):
- ✅ POST `/management/products` - принимает price
- ✅ PUT `/management/products/:id` - обновляет price

### 3. Frontend - Управление (`management.js`, `index.html`):
- ✅ Добавлено поле "Цена за тонну" в форму товаров
- ✅ Цена отображается в таблице товаров
- ✅ Цена сохраняется и редактируется

### 4. Frontend - Расход (`expense.js`):
- ✅ При выборе товара автоматически подставляется цена из справочника
- ✅ Пользователь может изменить цену вручную
- ✅ Автоматический пересчет суммы после подстановки цены

## ВАЖНО! Миграция базы данных:

Перед использованием выполните SQL скрипт:

```bash
psql -h switchyard.proxy.rlwy.net -p 15057 -U postgres -d railway < postgres-version/database/add-product-price.sql
```

Или выполните вручную в Railway Dashboard:
```sql
ALTER TABLE products 
ADD COLUMN IF NOT EXISTS price DECIMAL(10, 2) DEFAULT 0;
```

## Как работает:

### Добавление цены товара:
1. Откройте раздел "Управление"
2. В форме "Товары" заполните:
   - Название товара
   - Вес единицы
   - **Цена за тонну** (новое поле)
3. Нажмите "Добавить"

### Автозаполнение в расходе:
1. Откройте раздел "Расход"
2. Выберите фирму и склад
3. Выберите товар из списка
4. **Цена автоматически подставится** из справочника
5. Можно изменить цену вручную если нужно

## Пример использования:

### Настройка цен:
1. Управление → Товары
2. Добавить "Цемент М400", цена: 150
3. Добавить "Цемент М500", цена: 180

### Использование в расходе:
1. Расход → Выбрать товар "Цемент М400"
2. Цена автоматически = 150
3. Ввести количество: 100
4. Тонны автоматически = 5 (100/20)
5. Сумма автоматически = 750 (150 × 5)
