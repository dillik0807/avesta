✅ МНОЖЕСТВЕННЫЙ ВЫБОР ГРУПП СКЛАДОВ ДЛЯ ПОЛЬЗОВАТЕЛЕЙ

📋 ЧТО ИЗМЕНЕНО:

1. ✅ Заменен select на чекбоксы
   - Вместо выпадающего списка теперь список чекбоксов
   - Можно выбрать несколько групп складов одновременно
   - Прокручиваемый блок (max-height: 192px)

2. ✅ Чекбокс "Все склады"
   - При выборе "Все склады" все остальные чекбоксы снимаются
   - При выборе любой группы "Все склады" снимается автоматически

3. ✅ Сохранение в базу данных
   - Если выбрано "Все склады" → warehouse_group = null
   - Если выбрана одна группа → warehouse_group = "Группа 1"
   - Если выбрано несколько групп → warehouse_group = ["Группа 1", "Группа 2"]

4. ✅ Отображение в таблице
   - Если массив групп → показывается через запятую: "Группа 1, Группа 2"
   - Если одна группа → показывается как есть: "Группа 1"
   - Если null → показывается "Все склады"

5. ✅ Редактирование пользователя
   - При редактировании правильно устанавливаются чекбоксы
   - Поддержка старого формата (одна группа как строка)
   - Поддержка нового формата (массив групп)

🎯 КАК РАБОТАЕТ:

При добавлении пользователя:
1. Выбираете роль "Завсклада"
2. Отмечаете нужные группы складов чекбоксами
3. Можно выбрать несколько групп
4. Или оставить "Все склады"

При редактировании:
1. Нажимаете ✏️ на пользователе
2. Чекбоксы автоматически устанавливаются
3. Можете изменить выбор
4. Сохраняете изменения
