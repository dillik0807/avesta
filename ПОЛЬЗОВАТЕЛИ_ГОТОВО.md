## 🔐 Безопасность

- ✅ Пароли хешируются с помощью bcrypt (10 раундов)
- ✅ Пароли никогда не возвращаются в API ответах
- ✅ Минимальная длина пароля: 4 символа
- ✅ Проверка текущего пароля при смене
- ✅ Защита от удаления администратора
- ✅ JWT авторизация для всех endpoints

## 🎯 Функционал

### Роли пользователей
- **Администратор** - полный доступ
- **Завсклада** - управление складом
- **Кассир** - работа с расходами и погашениями
- **Руководитель** - просмотр отчетов

### Группы складов
- Можно ограничить доступ пользователя к определенной группе складов
- Если группа не указана - доступ ко всем складам
- Группы берутся из справочника складов

### Операции
1. ✅ Добавление пользователя
2. ✅ Редактирование пользователя
3. ✅ Удаление пользователя (кроме admin)
4. ✅ Смена пароля (с проверкой текущего)
5. ✅ Просмотр списка пользователей

## 📊 База данных

Таблица `users` уже существует в schema.sql:
```sql
CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    username VARCHAR(50) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    role VARCHAR(20) NOT NULL,
    warehouse_group VARCHAR(100)
);
```

Пользователь по умолчанию:
- Логин: `admin`
- Пароль: `admin123`
- Роль: `admin`

## 🚀 Как использовать

1. Откройте http://localhost:8080
2. Войдите как admin / admin123
3. Перейдите в раздел "👤 Пользователи"
4. Добавьте новых пользователей
5. Настройте роли и группы складов

## ✅ Тестирование

### Проверьте следующее:

1. **Добавление пользователя**
   - Заполните форму
   - Нажмите "Добавить пользователя"
   - Пользователь появится в таблице

2. **Редактирование**
   - Нажмите ✏️ рядом с пользователем
   - Измените данные
   - Сохраните изменения

3. **Удаление**
   - Нажмите 🗑️ рядом с пользователем (не admin)
   - Подтвердите удаление
   - Пользователь удалится

4. **Смена пароля**
   - Выберите пользователя
   - Введите текущий и новый пароль
   - Нажмите "Изменить пароль"

5. **Группы складов**
   - Селект должен заполниться группами из справочника складов
   - Можно выбрать группу или оставить "Все склады"

## 📝 Примечания

1. Пароль администратора по умолчанию - `admin123`. Рекомендуется сменить его после первого входа.

2. При редактировании пользователя:
   - Если оставить поле пароля пустым - старый пароль сохранится
   - Если ввести новый пароль - он будет обновлен

3. Администратор может:
   - Видеть всех пользователей
   - Менять пароль любому пользователю
   - Удалять пользователей (кроме себя)

4. Обычные пользователи могут:
   - Менять только свой пароль
   - Видеть только себя в списке смены пароля
