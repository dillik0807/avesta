КАК РАБОТАЕТ:

1. УСТАНОВКА ЦЕН:
   - Админ выбирает товар
   - Выбирает группу складов (A, B, C или "Все склады")
   - Устанавливает цену
   - Цена сохраняется для этой комбинации

2. АВТОЗАПОЛНЕНИЕ В РАСХОДЕ:
   - Пользователь выбирает склад
   - Система определяет группу склада
   - Выбирает товар
   - Система ищет цену:
     а) Сначала для конкретной группы склада
     б) Если не найдена - для "Все склады"
     в) Если не найдена - из товара
   - Цена автоматически заполняется

ГРУППЫ СКЛАДОВ:

- 🌍 Все склады (ALL) - цена для всех складов
- Группа A - своя цена
- Группа B - своя цена  
- Группа C - своя цена

✅ ПРОВЕРКА:

1. Откройте "Управление"
2. Установите цену для товара:
   - Товар: Навои Карбамид
   - Группа: Группа A
   - Цена: 200
   
3. Установите другую цену:
   - Товар: Навои Карбамид
   - Группа: Группа B
   - Цена: 250

4. Перейдите в "Расход"
5. Выберите склад из группы A
6. Выберите товар - цена должна быть 200
7. Выберите склад из группы B
8. Цена должна измениться на 250
