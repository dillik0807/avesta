Функционал:
===========

1. Фильтрация:
   - По периоду (дата с / дата по)
   - По клиенту (выбор из списка или все клиенты)
   - По умолчанию показывается текущий год

2. Отображение:
   - Таблица со всеми погашениями
   - Сортировка по дате (новые сначала)
   - Итоговые суммы отдельно в Сомони и Долларах

3. Печать:
   - Открывается новое окно с отформатированным отчетом
   - Включает заголовок, фильтры, таблицу и итоги
   - Оптимизировано для печати на A4

4. Экспорт в Excel:
   - Экспорт всех данных отчета
   - Имя файла: Отчет_по_погашениям_[дата_с]_[дата_по].xlsx
   - Включает все колонки таблицы

Как использовать:
=================

1. Перейти в раздел "💰 Отчёт по погашениям" в меню
2. Выбрать фильтры:
   - Период с (по умолчанию - начало года)
   - Период по (по умолчанию - конец года)
   - Клиент (по умолчанию - все клиенты)
3. Нажать "📊 Сформировать отчет"
4. Использовать кнопки:
   - 🖨️ Печать - для печати отчета
   - 📊 Excel - для экспорта в Excel

Особенности:
============

- Отчет показывает все погашения за выбранный период
- Суммы группируются по валютам (Сомони и Доллар)
- Можно фильтровать по конкретному клиенту
- Данные сортируются по дате (новые сначала)
- Печать и Excel включают все отфильтрованные данные
