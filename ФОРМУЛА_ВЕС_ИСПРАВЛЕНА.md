# ✅ Формула расчета веса в приходе исправлена

## Изменение:
Изменена формула расчета "Вес Тонн Факт" в форме "Приход".

## Было:
```
Вес Тонн Факт = Коли-по док / 20
```

## Стало:
```
Вес Тонн Факт = Коли-Факт / 20
```

## Логика:
Теперь вес рассчитывается из фактического количества (Коли-Факт), а не из документального (Коли-по док).

## Как работает:

1. Пользователь вводит "Коли-по док" (количество по документам)
2. Пользователь вводит "Коли-Факт" (фактическое количество)
3. Автоматически рассчитывается:
   - **Разница** = Коли-Факт - Коли-по док
   - **Вес Тонн Факт** = Коли-Факт / 20

## Пример:

### Входные данные:
- Коли-по док: 1000
- Коли-Факт: 980

### Результат:
- Разница: -20 (недостача)
- Вес Тонн Факт: 49.000 тонн (980 / 20)

## Измененные файлы:
1. `postgres-version/frontend/js/income.js`:
   - Функция `calculateIncomeWeight()` - изменена формула
   - Event listener - теперь вес пересчитывается при изменении "Коли-Факт"

## Как проверить:

1. Откройте форму "Приход"
2. Введите в "Коли-по док": 1000
3. Введите в "Коли-Факт": 980
4. Проверьте результаты:
   - Разница: -20
   - Вес Тонн Факт: 49.000

5. Измените "Коли-Факт" на 1020
6. Проверьте что вес пересчитался: 51.000

## Важно:
Вес теперь зависит только от фактического количества, что логично - мы взвешиваем то, что реально получили, а не то, что указано в документах.
