# 🔍 Отладка автозаполнения цены

## Проблема:
Цена не устанавливается автоматически при выборе товара в форме "Расход".

## Что исправлено:

1. **Изменены события** - теперь используется `blur` вместо `input` для корректной работы с datalist
2. **Добавлено подробное логирование** - можно отследить весь процесс в консоли

## Как проверить:

### Шаг 1: Проверка миграции БД
Откройте Railway Dashboard и выполните:
```sql
SELECT column_name, data_type 
FROM information_schema.columns 
WHERE table_name = 'products';
```

Должно быть поле `price` типа `numeric`.

Если поля нет, выполните:
```sql
ALTER TABLE products 
ADD COLUMN IF NOT EXISTS price DECIMAL(10, 2) DEFAULT 0;
```

### Шаг 2: Добавление товара с ценой
1. Откройте раздел "Управление"
2. В форме "Товары" добавьте:
   - Название: "Тестовый товар"
   - Вес: 0.050
   - **Цена за тонну: 100**
3. Нажмите "Добавить"
4. Проверьте что в таблице товаров отображается цена "100 $"

### Шаг 3: Проверка загрузки данных
1. Откройте консоль браузера (F12)
2. Выполните:
```javascript
console.log('Товары:', window.appData.products);
```
3. Проверьте что у товаров есть поле `price`

Если поля `price` нет, перезагрузите страницу.

### Шаг 4: Тест автозаполнения
1. Откройте раздел "Расход"
2. Откройте консоль браузера (F12)
3. Выберите фирму и склад
4. Начните вводить "Тестовый" в поле "Товар"
5. Выберите "Тестовый товар" из списка
6. **Кликните вне поля** или нажмите Tab

В консоли должны появиться сообщения:
```
🔍 Автозаполнение цены для товара: Тестовый товар
📦 Всего товаров в справочнике: X
✅ Товар найден: {name: "Тестовый товар", price: 100, ...}
💰 Устанавливаем цену: 100
```

### Шаг 5: Если не работает

#### Проверка 1: Данные загружены?
```javascript
console.log('appData:', window.appData);
console.log('products:', window.appData?.products);
```

#### Проверка 2: Товар есть в списке?
```javascript
const product = window.appData.products.find(p => p.name === 'Тестовый товар');
console.log('Найденный товар:', product);
console.log('Цена товара:', product?.price);
```

#### Проверка 3: Ручной вызов функции
```javascript
// Введите название товара в поле
document.getElementById('expenseProduct').value = 'Тестовый товар';
// Вызовите функцию вручную
autofillProductPrice();
```

## Возможные причины проблемы:

1. **Миграция не выполнена** - поле price отсутствует в БД
2. **Сервер не перезапущен** - старый код на сервере
3. **Данные не перезагружены** - нужно обновить страницу
4. **Товар добавлен без цены** - цена = 0 или NULL
5. **Название не совпадает** - опечатка в названии товара

## Решение:

1. Выполните миграцию БД
2. Перезапустите backend сервер
3. Обновите страницу в браузере (Ctrl+F5)
4. Добавьте товар с ценой > 0
5. Проверьте в консоли что данные загружены
6. Выберите товар и кликните вне поля
