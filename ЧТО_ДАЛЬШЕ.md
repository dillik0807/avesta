## 🔧 Как добавлять функциональность:

### Пример: Добавить таблицу прихода

1. Откройте `postgres-version/frontend/index.html`
2. Найдите раздел `<div id="income" class="content-section">`
3. Добавьте HTML для формы и таблицы
4. Создайте файл `postgres-version/frontend/js/income.js`
5. Добавьте функции:
   - `loadIncomeTable()` - загрузка данных
   - `addIncome()` - добавление записи
   - `editIncome(id)` - редактирование
   - `deleteIncome(id)` - удаление
6. Подключите файл в `index.html`: `<script src="js/income.js"></script>`

### API уже готово!

Все эндпоинты уже работают:
- `POST /api/income` - добавить приход
- `GET /api/data/:year` - получить данные
- `PUT /api/income/:id` - обновить приход
- `DELETE /api/income/:id` - удалить приход

То же самое для expense, payments, management.

## 💡 Совет:

Копируйте функциональность из оригинального `index.html` (Firebase версия) и адаптируйте под PostgreSQL API через `window.api.*` методы.

Адаптер уже настроен, поэтому многие функции будут работать автоматически!
