ЧТО СДЕЛАНО:
------------

✅ Год по умолчанию сохраняется в базе данных PostgreSQL
✅ При входе автоматически загружается год с галочкой
✅ Селектор в боковом меню показывает правильный год
✅ Можно переключаться между годами в любое время
✅ Галочка не теряется после закрытия браузера

═══════════════════════════════════════════════════════════════

ЗАПУСК (3 ПРОСТЫХ ШАГА):
-------------------------

1. МИГРАЦИЯ БАЗЫ ДАННЫХ:
   
   cd postgres-version/backend
   node scripts/add-user-settings-table.js
   
   Вы увидите:
   ✅ Таблица user_settings создана
   ✅ Индекс создан
   ✅ Миграция успешно выполнена!

2. ПЕРЕЗАПУСК СЕРВЕРА:
   
   Ctrl+C (остановить)
   npm start (запустить)
   
   Сервер должен запуститься на порту 3001

3. ГОТОВО!
   
   Откройте браузер и войдите в систему

КАК ИСПОЛЬЗОВАТЬ:
-----------------

УСТАНОВКА ГОДА ПО УМОЛЧАНИЮ:
-----------------------------
1. Войдите как администратор (admin / admin123)
2. Перейдите в раздел "👤 Пользователи"
3. Прокрутите вниз до "📅 Управление годами"
4. Поставьте галочку на нужный год (например, 2025)
5. В консоли появится: "✅ Год по умолчанию установлен в БД: 2025"

ПРОВЕРКА:
---------
1. Закройте браузер полностью
2. Откройте браузер снова
3. Войдите в систему
4. В консоли появится: "📅 Загружен год по умолчанию из БД: 2025"
5. В боковом меню селектор покажет: [2025 ▼]
6. Отображение: "Текущий: 2025"
7. Данные загрузятся для 2025 года

ПЕРЕКЛЮЧЕНИЕ ГОДОВ:
-------------------

Способ 1 - Боковое меню:
  • Нажмите на селектор "Рабочий год"
  • Выберите нужный год
  • Данные автоматически обновятся

Способ 2 - Раздел "Пользователи":
  • Перейдите в "Пользователи"
  • Найдите "Управление годами"
  • Нажмите кнопку "Переключить" у нужного года
  • Данные автоматически обновятся

ВАЖНО ПОНИМАТЬ:
---------------

ГОД ПО УМОЛЧАНИЮ ≠ ТЕКУЩИЙ РАБОЧИЙ ГОД

• Год по умолчанию (с галочкой):
  - Загружается автоматически при входе
  - Сохраняется в базе данных
  - Не меняется при переключении годов

• Текущий рабочий год:
  - С которым вы работаете сейчас
  - Можно менять в любое время
  - Не влияет на год по умолчанию
