### Что нужно изменить в скопированном коде:

#### 1. Замените Firebase вызовы на API вызовы:

**Было (Firebase):**
```javascript
const { ref, set, get } = window.firebaseRefs;
const dataRef = ref(window.firebaseDB, 'retailAppData');
await set(dataRef, data);
```

**Стало (PostgreSQL API):**
```javascript
await window.api.addIncome(data);
// или
await window.api.addExpense(data);
// или
await window.api.addPayment(data);
```

#### 2. Замените функции сохранения:

**Было:**
```javascript
function saveData() {
    localStorage.setItem('retailAppData', JSON.stringify(appData));
}
```

**Стало:**
```javascript
async function saveData() {
    // Данные автоматически сохраняются через API
    // при добавлении/редактировании записей
}
```

#### 3. Замените функции загрузки:

**Было:**
```javascript
function loadData() {
    const data = localStorage.getItem('retailAppData');
    appData = JSON.parse(data);
}
```

**Стало:**
```javascript
async function loadData() {
    const year = currentYear || 2025;
    const data = await window.api.getData(year);
    appData = data;
}
```

### Готовый шаблон адаптации:
