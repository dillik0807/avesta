# 📝 Пошаговая инструкция: Как выполнить миграцию базы данных

## Способ 1: Через Railway Dashboard (САМЫЙ ПРОСТОЙ)

### Шаг 1: Откройте Railway
1. Откройте браузер
2. Перейдите на https://railway.app/
3. Войдите в свой аккаунт

### Шаг 2: Найдите ваш проект
1. На главной странице найдите ваш проект с базой данных
2. Кликните на проект

### Шаг 3: Откройте PostgreSQL
1. В проекте найдите сервис "Postgres" или "PostgreSQL"
2. Кликните на него

### Шаг 4: Откройте Query Editor
1. В верхнем меню найдите вкладку "Query" или "Data"
2. Кликните на неё
3. Откроется окно для ввода SQL команд

### Шаг 5: Вставьте команду
1. В поле для ввода SQL вставьте эту команду:

```sql
ALTER TABLE products ADD COLUMN IF NOT EXISTS price DECIMAL(10, 2) DEFAULT 0;
```

2. Нажмите кнопку "Run" или "Execute" или нажмите Ctrl+Enter

### Шаг 6: Проверьте результат
Должно появиться сообщение "ALTER TABLE" или "Success"

### Шаг 7: Установите цену для товара
Выполните ещё одну команду:

```sql
UPDATE products SET price = 150 WHERE name = 'Навои Карбамид';
```

Нажмите "Run"

### Шаг 8: Проверьте что всё работает
Выполните:

```sql
SELECT * FROM products;
```

Вы должны увидеть колонку "price" со значением 150

## Способ 2: Через код (автоматически)

Я могу создать скрипт который выполнит миграцию автоматически.

### Шаг 1: Создам скрипт миграции
Скрипт уже создан: `postgres-version/database/add-product-price.sql`

### Шаг 2: Запустите скрипт
В терминале выполните:

```bash
cd postgres-version/backend
node -e "
const db = require('./db');
(async () => {
  try {
    await db.query('ALTER TABLE products ADD COLUMN IF NOT EXISTS price DECIMAL(10, 2) DEFAULT 0');
    console.log('✅ Миграция выполнена!');
    await db.query('UPDATE products SET price = 150 WHERE name = \\'Навои Карбамид\\'');
    console.log('✅ Цена установлена!');
    process.exit(0);
  } catch (error) {
    console.error('❌ Ошибка:', error);
    process.exit(1);
  }
})();
"
```
