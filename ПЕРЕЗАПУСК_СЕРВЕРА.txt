🔄 ПЕРЕЗАПУСК BACKEND СЕРВЕРА

ПРОБЛЕМА:
- Новый маршрут /api/prices не загружен
- Нужно перезапустить сервер

ЧТО ИСПРАВЛЕНО:
✅ API клиент теперь вызывает /prices вместо /management/prices
✅ Функция handleAddPrice передает правильный объект данных

ИНСТРУКЦИЯ:

1. Остановите backend сервер:
   - Нажмите Ctrl+C в терминале где запущен сервер

2. Запустите сервер заново:
   cd postgres-version/backend
   node server.js

3. Обновите браузер:
   - Нажмите Ctrl+F5 (полная перезагрузка)

4. Проверьте:
   - Откройте раздел "Управление"
   - Попробуйте добавить цену через "Управление ценами"
   - Цена должна сохраниться
   - Перейдите в "Расход"
   - Выберите товар - цена должна автоматически заполниться

ОЖИДАЕМЫЙ РЕЗУЛЬТАТ:
✅ Цены сохраняются в базу данных
✅ При выборе товара в расходе цена автоматически заполняется
✅ Пользователь может изменить цену вручную
