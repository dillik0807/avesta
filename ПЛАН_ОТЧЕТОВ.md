## Структура файлов:

Для каждого отчета создать:
1. JavaScript модуль: `js/[название]-report.js`
2. HTML секция в `index.html`
3. Подключение скрипта в `<head>`
4. Обработчик в `reports.js`

## Общий подход:

Все отчеты будут работать по единой схеме:
1. Кнопка в разделе "Отчёты" открывает отдельную секцию
2. Кнопка "← Назад к списку отчетов" для возврата
3. Фильтры с автозаполнением из справочников
4. Кнопки: Сформировать отчет, Печать, Excel
5. Таблица с итогами

## Приоритет реализации:

1. Отчёт по расходу товаров (самый востребованный)
2. Отчёт по приходу товаров
3. Расход по клиентам
4. Отчёт по складам
