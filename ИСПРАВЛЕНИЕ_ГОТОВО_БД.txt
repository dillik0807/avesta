ПРОБЛЕМА:
---------
Галочка года по умолчанию сохранялась в localStorage браузера
и исчезала после закрытия приложения или при входе с другого
устройства/браузера.

РЕШЕНИЕ:
--------
Теперь год по умолчанию сохраняется в базе данных PostgreSQL
в таблице user_settings. Каждый пользователь имеет свои
персональные настройки, которые сохраняются на сервере.

КАК ЗАПУСТИТЬ:
--------------

1. Откройте терминал в папке backend:
   cd postgres-version/backend

2. Запустите миграцию:
   node scripts/add-user-settings-table.js

3. Перезапустите сервер:
   Ctrl+C (остановить)
   npm start (запустить)

4. Готово! Теперь можно проверять.

═══════════════════════════════════════════════════════════════

КАК ПРОВЕРИТЬ:
--------------

1. Войдите в систему (admin / admin123)
2. Перейдите в раздел "Пользователи"
3. Прокрутите до "Управление годами"
4. Поставьте галочку на любой год
5. В консоли должно появиться:
   ✅ Год по умолчанию установлен в БД: XXXX
6. Закройте браузер полностью
7. Откройте браузер снова
8. Войдите в систему
9. В консоли должно появиться:
   📅 Загружен год по умолчанию из БД: XXXX
10. Перейдите в "Пользователи"
11. Галочка должна быть на том же году ✓

ТЕХНИЧЕСКИЕ ДЕТАЛИ:
-------------------

Таблица user_settings:
- user_id: ID пользователя (уникальный)
- default_year: Год по умолчанию
- settings: JSON для дополнительных настроек
- created_at: Дата создания
- updated_at: Дата обновления

API Endpoints:
- GET /api/settings - получить настройки
- POST /api/settings/default-year - установить год
- DELETE /api/settings/default-year - удалить год
- PUT /api/settings - обновить настройки
