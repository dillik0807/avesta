✅ АВТОЗАПОЛНЕНИЕ КЛИЕНТОВ С ПОИСКОМ - ГОТОВО

Поле "Клиент" в разделах Погашения и Партнеры:
- Используется: input + datalist (автозаполнение с поиском)
- При наборе текста появляется выпадающий список
- Список фильтруется по введенному тексту
- Показывает имя и телефон: "Имя (телефон)"

ИЗМЕНЕНИЯ:

1. index.html:
   - Погашения: input + datalist
   - Партнеры: input + datalist

2. payments.js:
   - loadPaymentDictionaries() заполняет datalist
   - Валидация проверяет существование клиента

3. partners.js:
   - loadPartnerDictionaries() заполняет datalist
   - Валидация проверяет существование клиента

Теперь при наборе имени клиента появляется выпадающий список с фильтрацией!
