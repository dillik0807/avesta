✅ ПРОВЕРКА СТАТУСА ПОЛЬЗОВАТЕЛЯ ПРИ ВХОДЕ

📋 ЧТО ДОБАВЛЕНО:

1. ✅ Проверка блокировки пользователя
   - Если is_blocked = true
   - Показывается сообщение: "🔒 Ваш аккаунт заблокирован. Обратитесь к администратору."
   - Вход в систему запрещен

2. ✅ Принудительная смена пароля
   - Если require_password_change = true
   - Показывается модальное окно с формой смены пароля
   - Пользователь не может продолжить работу без смены пароля
   - После смены пароля флаг автоматически сбрасывается

🎯 КАК РАБОТАЕТ:

Сценарий 1: Заблокированный пользователь
=========================================
1. Пользователь вводит логин и пароль
2. Система проверяет is_blocked
3. Если заблокирован → показывается ошибка
4. Вход запрещен

Сценарий 2: Требуется смена пароля
===================================
1. Пользователь вводит логин и пароль
2. Система проверяет require_password_change
3. Если требуется → показывается модальное окно
4. Пользователь вводит новый пароль (дважды)
5. Пароль меняется, флаг сбрасывается
6. Вход продолжается автоматически

Сценарий 3: Обычный вход
========================
1. Пользователь вводит логин и пароль
2. Проверки пройдены
3. Вход в систему

🔐 МОДАЛЬНОЕ ОКНО СМЕНЫ ПАРОЛЯ:

Заголовок: 🔑 Требуется смена пароля
Текст: "Администратор требует, чтобы вы сменили пароль перед продолжением работы."

Поля:
- Новый пароль (минимум 4 символа)
- Подтвердите пароль

Кнопки:
- "Сменить пароль" (синяя) - меняет пароль и продолжает вход
- "Выйти" (серая) - перезагружает страницу

Валидация:
- Пароли должны совпадать
- Минимум 4 символа
- Показывается ошибка если что-то не так

📊 ЛОГИКА:

```javascript
if (user.is_blocked) {
    // Показать ошибку "Заблокирован"
    return;
}

if (user.require_password_change) {
    // Показать модальное окно смены пароля
    // После смены → продолжить вход
    return;
}

// Обычный вход
```

🧪 ТЕСТИРОВАНИЕ:

Тест 1: Блокировка
==================
1. Войдите как admin
2. Перейдите в "Управление" → "Пользователи"
3. Создайте тестового пользователя
4. Установите галочку "Заблокировать пользователя"
5. Сохраните
6. Выйдите из системы
7. Попробуйте войти как тестовый пользователь
8. Должна появиться ошибка: "🔒 Ваш аккаунт заблокирован"

Тест 2: Смена пароля
====================
1. Войдите как admin
2. Создайте тестового пользователя
3. Установите галочку "Требуется смена пароля при входе"
4. Сохраните
5. Выйдите из системы
6. Войдите как тестовый пользователь
7. Должно появиться модальное окно смены пароля
8. Введите новый пароль (дважды)
9. Нажмите "Сменить пароль"
10. Должен произойти вход в систему
11. Бейдж "🔑 Смена пароля" должен исчезнуть

Тест 3: Оба флага
=================
1. Установите оба флага (блокировка + смена пароля)
2. Попробуйте войти
3. Блокировка имеет приоритет
4. Должна показаться ошибка блокировки
