✅ ВСЕ ОТЧЕТЫ ГОТОВЫ

Создано 5 новых отчетов:

1. 📅 Отчёт за текущий день (daily-report.js)
   - Показывает приходы и расходы товаров за выбранную дату
   - Две таблицы: Приходы (с вагонами) и Расходы (с клиентами)
   - Приходы: Вагон, Фирма, Склад, Товар, По док, Факт, Разница, Тонн
   - Расходы: Фирма, Склад, Товар, Клиент, Количество, Тонн, Цена, Сумма
   - Кнопка в отчетах: "📅 Отчёт за текущий день" (data-report="daily-report")

2. 📤 Отчёт по расходу товаров (expense-report.js)
   - Фильтры: Период с/по, Склад, Товар, Клиент
   - Колонки: Дата, Фирма, Склад, Товар, Клиент, Количество, Тонн, Цена, Сумма, Пользователь
   - Итоги: Количество, Тонн, Сумма
   - Кнопка в отчетах: "📤 Отчёт по расходу товаров" (data-report="product-expense")

3. 📥 Отчёт по приходу товаров (income-report.js)
   - Фильтры: Период с/по, Склад, Товар, Фирма
   - Колонки: Дата, Вагон, Фирма, Склад, Товар, По док, Факт, Разница, Тонн, Пользователь
   - Итоги: По док, Факт, Тонн
   - Разница с цветовой индикацией (зеленый/красный)
   - Кнопка в отчетах: "📥 Отчёт по приходу товаров" (data-report="product-income")

4. 👤 Расход по клиентам (client-report.js)
   - Фильтры: Период с/по, Клиент, Склад
   - Колонки: Дата, Коалица, Номер, Клиент, Товар, Фирма, Склад, Количество, Вес тонн, Цена, Сумма, Примечания, Пользователь
   - Итоги: Количество, Тонн, Сумма
   - Детальный отчет по операциям (не сводка)
   - Кнопка в отчетах: "👤 Расход по клиентам" (data-report="client-expense")

5. 🏢 Отчёт по складам (warehouse-report.js)
   - Фильтры: Период с/по, Склад, Товар
   - Колонки: Фирма, Склад, Товар, Приход, Расход, Остаток, Остаток тонн
   - Итоги: Приход, Расход, Остаток, Остаток тонн
   - Цветовая индикация остатков: зеленый (положительный), серый (нулевой), красный (отрицательный)
   - Кнопка в отчетах: "🏢 Отчёт по складам" (data-report="warehouse-report")

- Таблицы с итогами в tfoot
