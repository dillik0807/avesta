✅ КНОПКИ УПРАВЛЕНИЯ СТАТУСОМ В ТАБЛИЦЕ

📋 ЧТО ДОБАВЛЕНО:

В таблице пользователей теперь 4 кнопки для каждого пользователя:

1. ✏️ Редактировать (синяя)
   - Открывает форму редактирования
   - Заполняет все поля текущими данными

2. 🔑 Смена пароля (желтая)
   - Быстро устанавливает/снимает флаг require_password_change
   - Переключается одним кликом
   - Показывает подтверждение

3. 🔒/🔓 Блокировка (красная/зеленая)
   - 🔒 Заблокировать (красная) - если пользователь активен
   - 🔓 Разблокировать (зеленая) - если пользователь заблокирован
   - Переключается одним кликом
   - Нельзя заблокировать admin

4. 🗑️ Удалить (серая)
   - Удаляет пользователя
   - Нельзя удалить admin

🎯 КАК РАБОТАЕТ:

Кнопка 🔑 (Смена пароля):
=========================
1. Нажимаете на кнопку
2. Появляется подтверждение: "🔑 Требовать смену пароля для пользователя X?"
3. Подтверждаете
4. Флаг устанавливается
5. Таблица обновляется
6. Появляется бейдж "🔑 Смена пароля"

Повторное нажатие:
1. Появляется подтверждение: "✅ Отменить смену пароля для пользователя X?"
2. Подтверждаете
3. Флаг снимается
4. Бейдж исчезает

Кнопка 🔒/🔓 (Блокировка):
===========================
Если пользователь активен (кнопка 🔒 красная):
1. Нажимаете на кнопку
2. Появляется подтверждение: "🔒 Заблокировать пользователя X?"
3. Подтверждаете
4. Пользователь блокируется
5. Кнопка меняется на 🔓 (зеленая)
6. Появляется бейдж "🔒 Заблокирован"

Если пользователь заблокирован (кнопка 🔓 зеленая):
1. Нажимаете на кнопку
2. Появляется подтверждение: "🔓 Разблокировать пользователя X?"
3. Подтверждаете
4. Пользователь разблокируется
5. Кнопка меняется на 🔒 (красная)
6. Бейдж исчезает

Цвета кнопок:
- ✏️ Синяя (редактировать)
- 🔑 Желтая (смена пароля)
- 🔒 Красная (заблокировать)
- 🔓 Зеленая (разблокировать)
- 🗑️ Серая (удалить)

🔐 ЗАЩИТА:

1. Нельзя заблокировать admin
   - Кнопка 🔒 работает, но показывает ошибку
   
2. Нельзя удалить admin
   - Кнопка 🗑️ неактивна (серая, полупрозрачная)

3. Все действия требуют подтверждения
   - Показывается диалог confirm()
   - Можно отменить

4. После каждого действия таблица обновляется
   - Данные загружаются с сервера
   - Бейджи обновляются автоматически

🧪 ТЕСТИРОВАНИЕ:

Тест 1: Установить смену пароля
================================
1. Найдите тестового пользователя в таблице
2. Нажмите кнопку 🔑 (желтая)
3. Подтвердите действие
4. Должен появиться бейдж "🔑 Смена пароля"
5. Выйдите и войдите под этим пользователем
6. Должно появиться окно смены пароля

Тест 2: Заблокировать пользователя
===================================
1. Найдите тестового пользователя в таблице
2. Нажмите кнопку 🔒 (красная)
3. Подтвердите действие
4. Кнопка должна стать 🔓 (зеленая)
5. Должен появиться бейдж "🔒 Заблокирован"
6. Выйдите и попробуйте войти под этим пользователем
7. Должна появиться ошибка "Аккаунт заблокирован"

Тест 3: Разблокировать пользователя
====================================
1. Найдите заблокированного пользователя
2. Нажмите кнопку 🔓 (зеленая)
3. Подтвердите действие
4. Кнопка должна стать 🔒 (красная)
5. Бейдж "🔒 Заблокирован" должен исчезнуть
6. Пользователь снова может войти
