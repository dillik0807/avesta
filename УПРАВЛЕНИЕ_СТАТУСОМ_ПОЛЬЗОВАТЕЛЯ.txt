✅ УПРАВЛЕНИЕ СТАТУСОМ ПОЛЬЗОВАТЕЛЯ

📋 ЧТО ДОБАВЛЕНО:

1. ✅ Галочка "Требуется смена пароля при входе"
   - Админ может установить эту галочку
   - Пользователь должен будет сменить пароль при следующем входе
   - Поле: require_password_change (boolean)

2. ✅ Галочка "Заблокировать пользователя"
   - Админ может заблокировать пользователя
   - Заблокированный пользователь не сможет войти в систему
   - Красный текст для привлечения внимания
   - Поле: is_blocked (boolean)

3. ✅ Колонка "Статус" в таблице пользователей
   - Показывает текущий статус пользователя
   - Цветные бейджи для визуального отображения

📊 СТАТУСЫ ПОЛЬЗОВАТЕЛЯ:

✅ Активен (зеленый бейдж)
   - Пользователь может войти
   - Смена пароля не требуется
   - Не заблокирован

🔑 Смена пароля (желтый бейдж)
   - require_password_change = true
   - Пользователь должен сменить пароль при входе
   - Может отображаться вместе с "Активен"

🔒 Заблокирован (красный бейдж)
   - is_blocked = true
   - Пользователь не может войти в систему
   - Приоритетный статус

🎯 КАК РАБОТАЕТ:

При добавлении пользователя:
1. Заполняете форму (логин, пароль, роль, группы)
2. Устанавливаете галочки по необходимости:
   ☐ Требуется смена пароля при входе
   ☐ Заблокировать пользователя
3. Нажимаете "Добавить пользователя"

При редактировании:
1. Нажимаете ✏️ на пользователе
2. Галочки автоматически устанавливаются
3. Можете изменить статус
4. Сохраняете изменения

В таблице:
- Колонка "Статус" показывает текущее состояние
- Несколько бейджей могут отображаться одновременно
- Цветовая индикация для быстрого распознавания

🔐 ЛОГИКА БЛОКИРОВКИ:

Заблокированный пользователь (is_blocked = true):
- ❌ Не может войти в систему
- ❌ Не может выполнять операции
- ✅ Остается в базе данных
- ✅ Админ может разблокировать

Требуется смена пароля (require_password_change = true):
- ✅ Может войти в систему
- ⚠️ Должен сменить пароль при входе
- ❌ Не может работать до смены пароля
- ✅ После смены пароля флаг сбрасывается

📐 ПРИМЕРЫ СТАТУСОВ:

Пользователь 1:
- is_blocked = false
- require_password_change = false
→ Статус: ✅ Активен

Пользователь 2:
- is_blocked = false
- require_password_change = true
→ Статус: 🔑 Смена пароля

Пользователь 3:
- is_blocked = true
- require_password_change = false
→ Статус: 🔒 Заблокирован

Пользователь 4:
- is_blocked = true
- require_password_change = true
→ Статус: 🔒 Заблокирован 🔑 Смена пароля
