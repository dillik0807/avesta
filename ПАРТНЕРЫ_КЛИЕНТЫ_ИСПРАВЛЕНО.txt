✅ ПАРТНЕРЫ - КЛИЕНТЫ ИЗ УПРАВЛЕНИЯ - ИСПРАВЛЕНО

Проблема: Клиенты не отображались в выпадающем списке партнеров
Решение: Добавлена загрузка справочников при переходе в раздел

ИЗМЕНЕНИЯ:

1. partners.js:
   - loadPartnerDictionaries() теперь глобальная функция (window.loadPartnerDictionaries)
   - Загружает клиентов из window.appData.clients
   - Показывает имя и телефон: "Имя (телефон)"

2. main.js:
   - При переходе в раздел "partners" вызывается loadPartnerDictionaries()
   - Справочники обновляются каждый раз при открытии раздела

3. adapter.js:
   - При загрузке данных вызывается loadPartnerDictionaries()
   - Справочники обновляются после получения данных из API

РЕЗУЛЬТАТ:
Теперь при открытии раздела "Партнеры" клиенты загружаются из таблицы 
"Клиенты" в разделе "Управление" и отображаются в выпадающем списке 
с автозаполнением при наборе текста.
