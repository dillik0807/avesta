=== ЗАПУСК МИГРАЦИИ УПРАВЛЕНИЯ ЦЕНАМИ ===

Выполните эту команду в терминале:

cd postgres-version/backend
node scripts/create-prices-table.js

Эта команда:
1. Создаст таблицу product_prices для хранения истории цен
2. Перенесет текущие цены из таблицы products
3. Создаст индексы для быстрого поиска

После выполнения:
- Перезапустите backend сервер
- Обновите страницу в браузере

Теперь система будет:
✅ Хранить историю всех изменений цен
✅ Автоматически подставлять последнюю цену при выборе товара
✅ Позволять админу управлять ценами через интерфейс
